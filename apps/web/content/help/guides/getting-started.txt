# Getting Started

Welcome to the Docs Marketing API! This guide will help you get up and running quickly.

## Prerequisites

Before you begin, make sure you have:

1. A Docs Marketing account (sign up at https://docs-marketing.com)
2. An API key (generate one in your account dashboard)
3. Basic knowledge of REST APIs and HTTP requests

## Quick start

Follow these steps to make your first API call:

### Step 1: Get your API key

1. Log in to your Docs Marketing account
2. Navigate to Settings > API Keys
3. Click "Generate New API Key"
4. Copy your API key and store it securely

**Important**: Never share your API key or commit it to version control!

### Step 2: Make your first request

Use your API key to authenticate requests by including it in the `Authorization` header:

```bash
curl https://api.docs-marketing.com/v1/users \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json"
```

### Step 3: Handle the response

Successful responses will have this format:

```json
{
  "success": true,
  "data": {
    "users": [
      {
        "id": "user-123",
        "name": "John Doe",
        "email": "john@example.com"
      }
    ]
  }
}
```

## Common use cases

### Sync user data

Keep your database in sync with Docs Marketing user data:

```javascript
async function syncUsers() {
  const response = await fetch('https://api.docs-marketing.com/v1/users', {
    headers: {
      Authorization: `Bearer ${API_KEY}`,
      'Content-Type': 'application/json',
    },
  });

  const { data } = await response.json();
  // Store users in your database
}
```

### Create a new user

```javascript
async function createUser(userData) {
  const response = await fetch('https://api.docs-marketing.com/v1/users', {
    method: 'POST',
    headers: {
      Authorization: `Bearer ${API_KEY}`,
      'Content-Type': 'application/json',
    },
    body: JSON.stringify(userData),
  });

  return response.json();
}
```
